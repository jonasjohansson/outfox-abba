ABBA Song Pack
==============

Add songs here. Each song needs its own folder with:

  SongName/
    song.ogg       - Audio file (Ogg Vorbis, ~192kbps recommended)
    song.ssc        - Step chart file (use OutFox's built-in editor to create)
    banner.png      - Song banner (418x164 px)
    background.png  - Background image (640x480 px or 1920x1080 px)
    jacket.png      - Square jacket art (optional, 256x256 px)

Example:
  Dancing Queen/
    song.ogg
    song.ssc
    banner.png
    background.png

To create a chart:
  1. Place your .ogg file in the song folder
  2. Open OutFox and go to the editor
  3. Select your song and create charts for desired difficulties
  4. Save - the .ssc file will be created automatically

Note: For distribution, use licensed covers or original arrangements
rather than copyrighted ABBA recordings.
